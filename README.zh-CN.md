# SugarLand 元宇宙25号
AI powered Sugarscape

## 项目总结

SugarLand 是经典 [Sugarscape](https://en.wikipedia.org/wiki/Sugarscape)（糖域）智能体建模（ABM）的 AI 驱动重新实现，原始模型由 Epstein 和 Axtell 在《人工社会》（1996）中提出。本项目用 **AI Agent** 替代传统的硬编码智能体，使模拟经济中的社会涌现行为更加丰富和真实。

### 核心特性

| 特性 | 描述 |
|---|---|
| **AI 智能体模拟** | 智能体由 AI 驱动，而非简单规则脚本，实现更真实的决策行为 |
| **双资源经济** | 糖与香料资源，具备季节周期、污染与再生机制 |
| **社会行为** | 易货贸易（看涨/看跌策略）、遗传繁殖与财富继承 |
| **50×50 环面网格** | 环绕拓扑消除边缘偏差；双峰资源地形驱动迁徙行为 |
| **人口动力学** | 出生、衰老、死亡、自然选择压力与涌现的财富不平等（基尼系数） |
| **实时监控** | 计分板展示人口统计、属性分布、财富基尼系数与性能指标 |

### 技术架构

- **细胞空间（Cellspace）**：50×50 网格，环面拓扑，双峰资源地形，季节性生长率与污染累积。
- **公民（Citizen）**：异构智能体，具有遗传决定的视觉（*v* ∈ [1,6]）、代谢率（*m* ∈ [1,4]）、最大年龄 ∈ [60,100]、初始财富 ∈ [5,25]。
- **规则引擎**：严格的 G→M→R 执行顺序（生长→移动→更替），智能体随机调度以消除先手偏差。
- **高级经济系统**：基于边际价值的易货贸易，包含看涨（高频小额）与看跌（安全边际）交易策略。

### 项目目标

本项目的实际意义在于提供一个易于使用的平台，通过 AI 驱动的仿真来探索关于财富不平等、市场动态与社会结构的问题——例如：*为什么富人富、穷人穷？*、*自由市场能解决什么、不能解决什么？*、*遗产和初始禀赋如何塑造社会分层？*
